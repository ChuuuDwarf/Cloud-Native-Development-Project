# 實驗室資訊管理系統（LIMS）功能規格 — 依功能細分角色

---

## 系統角色定義 (role)

| 角色 | 簡稱 | 說明 |
|------|------|------|
| 廠區使用者 | 使用者 | 提出送測需求的廠區端人員 |
| 實驗室人員 | 實驗員 | 執行收樣、派工、實驗的操作人員 |

| 系統管理者 | 管理者 | 維護系統設定與帳號的後台人員 |

```
草稿
  └─► 待簽核
        ├─► 退回補件 ──► (修改後) 待簽核
        ├─► 已拒絕
        ├─► 已取消
        └─► 已核准
              └─► 待送樣
                    └─► 已收樣
                          └─► 已分貨
                                └─► 排程中
                                      └─► 實驗中
                                            ├─► 暫停中
                                            │     └─► 待主管判定
                                            │           ├─► 已終止
                                            │           └─► 實驗中（繼續）
                                            └─► 待結果確認
                                                  └─► 已完成
                                                        └─► 待報告回傳
                                                              └─► 待取件
                                                                    └─► 已結案
```
