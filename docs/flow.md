## 委託單完整狀態流程

```
草稿
  └─► 待簽核
        ├─► 退回補件 ──► (修改後) 待簽核
        ├─► 已拒絕
        ├─► 已取消
        └─► 已核准
              └─► 待送樣
                    └─► 已收樣
                          └─► 已分貨
                                └─► 排程中
                                      └─► 實驗中
                                            ├─► 暫停中
                                            │     └─► 待主管判定
                                            │           ├─► 已終止
                                            │           └─► 實驗中（繼續）
                                            └─► 待結果確認
                                                  └─► 已完成
                                                        └─► 待報告回傳
                                                              └─► 待取件
                                                                    └─► 已結案
```
